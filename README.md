# Bluestock Mutual Fund Analytics - Capstone Project 1

## Project Overview
End-to-end mutual fund analytics pipeline built during Bluestock Fintech internship.

## Folder Structure
- data/raw - Original CSV datasets
- data/processed - Cleaned CSV files
- notebooks/ - Jupyter analysis notebooks
- scripts/ - Python automation scripts
- sql/ - Database schema and queries
- dashboard/ - Power BI dashboard
- reports/ - Final report and presentation

## Tech Stack
- Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly
- SQLite, SQLAlchemy
- Power BI
- Git/GitHub

## Key Deliverables
- ETL Pipeline fetching live NAV data
- SQLite database with 10 tables
- EDA with 9 professional charts
- Advanced risk metrics (VaR, CVaR, Sharpe)
- Interactive Power BI dashboard
